# Sblocca-il-contatore-segreto
🎮 Sblocca il contatore segreto

Un progetto interattivo sviluppato con **HTML, CSS e JavaScript** che simula un sistema di login per sbloccare un contatore.

---

🚀 Funzionalità

- 🔐 Sistema di login con validazione
- ⛔ Pulsanti bloccati fino all’accesso
- ⏱ Contatore incrementale
- 🕒 Orologio digitale in tempo reale
- 🎨 Interfaccia responsive e pulita

---

🧠 Obiettivo didattico

Questo progetto serve a praticare:

- Manipolazione DOM
- Eventi JavaScript
- Condizioni (`if/else`)
- Timer (`setInterval`)
- Gestione form
- CSS responsive
